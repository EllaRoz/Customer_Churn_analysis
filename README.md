# Customer_Churn_analysis
A Data Science and Analytics project using Python and Tableau

### Project Goal
The goal of this project is to analyze customer churn for a telecom company. With the help of a machine learning model a high-risk customers will be identified.

### Tools & Techniques
#### Python
- pandas, numpy
- scikit-learn
- XGBoost
- Jupyter Notebook

#### Tableau
- Interactive dashboard for the customer churn insights
- Cohort analysis

### Dataset
The project uses the Telco Customer Churn dataset from IBM.

Rows: 7,043 customers
Columns: 21 features + churn label

#### Features

**Demographics**
- Gender  
- Senior citizen  
- Partner  
- Dependents  

**Services**
- Phone service  
- Multiple lines  
- Internet service  
- Online security  
- Online backup  
- Device protection  
- Tech support  
- Streaming TV  
- Streaming movies  

**Account Information**
- Tenure  
- Contract type  
- Payment method  

**Financials**
- Monthly charges  
- Total charges  

**Target Variable**
- **Churn** (Yes/No)
